About Me

About

Hi, I'm Jack! I'm a student at the university of Plymouth, studying Computing & Games Development, with an interest in full-stack development. I have three years of experience developing single player games in 2D and 3D within the Unity Engine, developing both 2D and 3D games throughout that time. I also have a year of basic Java experience, developing mock applications for real world use cases such as a patient management system.

University Projects

Damp and Mould App

A client project for Plymouth Energy Community, we were tasked with creating a product that could easily convey issues surrounding Damp and Mould to homeowners.

I was the project lead, tasked with managing the product, UX and Code Review.

Dhelve

A procedurally generated dungeon crawler developed in Unity, where the player could progress through dungeons.

I was the technical lead, tasked with maintaining the Github repository, doing Code Review, and was the main systems programmer.

Highway Hero

An endless runner developed in Unity where the player aims to beat their highschore by reaching further distances.

As this was a solo project, I dealt with everything - I managed the project using Scrum, developed all system and sourced all models.

Necromancer-Game VR

A mixture between RPG and Tower Defence elements in VR, the player has to interact with villagers to secure information regarding body parts they can use to create units.

As this is a solo project, I'm dealing with all systems and project management.

Bloodletting

A Top Down Zombie Shooter created in my first semester of university, it introduced me to C#, serialization, and game development within the Unity Game Engine.

As this is a solo project, I dealt with all systems and project management.
